# 项目名称

- MarkVision

## 项目简介

- 一个简单易用的Markdown预览和编辑工具。

## 参考网站

- https://markdown.com.cn/editor/

## 目标：

- 将 https://markdown.com.cn/editor 封装为具备基础功能的Windows桌面应用

## 实现功能

- exe可执行文件一键启动
- 实时Markdown预览
- 支持选中文本同步高亮
- 滚动同步
- 导出为PDF

## 运行平台

- Windows11

## 技术栈

- Tauri（Rust + 系统WebView2）
